# v0.1.2-PRERELEASE

## Fixes

- Corrected installation error where `settings.json` was not created on first run.
- Switched from handrolled parser to [`wtf_wikipedia`](https://www.npmjs.com/package/wtf_wikipedia) for more robust template parsing.

## Features

- Added option to remove dates from appearance lists.
- Added option to save as a markdown (.md) list in addition to saving as a JSON.
